GeometryLibrary Methods

The GeometryLibrary type exposes the following members.

Methods

  Name Description
Angle
Returns the angle [radians] between the two vectors.
Area
Returns the area between two vectors.
ConcaveInside
True: The concave side of the vector is inside the shape. This is determined by the direction of the vector.
ConcavityColinearity
Returns a value indicating the concavity of the vectors. 1 = Pointing the same way. > 0 = Concave. 0 = Orthogonal. < 0 = Convex. -1 = Pointing the exact opposite way.
ConvexInside
True: The convex side of the vector is inside the shape. This is determined by the direction of the vector.
Cross
Cross-product of two vectors.
Dot
Dot product of two vectors.
IsColinearOppositeDirection
True: Segments are parallel, on the same line, oriented in the opposite direction.
IsColinearSameDirection
True: Segments are parallel, on the same line, oriented in the same direction.
IsConcave
Vectors form a concave angle.
IsConvex
Vectors form a convex angle.
IsOrthogonal
Vectors form a 90 degree angle.
NormalVector
Returns a normal vector to a line connecting two points.
TangentVector
Returns the tangent vector to the supplied points.
